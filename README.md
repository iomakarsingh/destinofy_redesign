# Destinofy AI - Modern Redesign 🚀

A stunning modern redesign of Destinofy.ai with cutting-edge UI/UX. Built with React, Tailwind CSS, and Framer Motion for smooth, engaging animations.

## ✨ Features

- **Modern Glass Morphism Design**: Contemporary frosted glass effect with gradient overlays
- **Smooth Animations**: Framer Motion animations for seamless user experience
- **Responsive Design**: Mobile-first approach, perfect on all devices
- **Interactive Elements**: Hover effects, micro-interactions, and dynamic backgrounds
- **Dark Theme**: Eye-friendly dark mode with gradient accents
- **Performance Optimized**: Fast load times and smooth 60fps animations
- **Accessible**: WCAG compliant with proper contrast and keyboard navigation

## 🎨 Design Highlights

### Colors & Gradients
- **Primary**: Blue to Cyan gradients for brand consistency
- **Background**: Dark slate with animated gradient orbs
- **Accents**: Yellow stars, green WhatsApp, multi-color highlights

### Components
- **Navigation**: Fixed header with mobile menu, glass effect
- **Hero Section**: Eye-catching gradient text, stats display, floating animations
- **How It Works**: 3-step process with animated connectors
- **Features**: 6 powerful features in responsive grid with hover effects
- **Use Cases**: 4 healthcare facility types with icon animations
- **Testimonials**: Social proof with rating stars and company info
- **FAQ**: Expandable questions with smooth animations
- **CTA**: Compelling call-to-action with animated backgrounds
- **Footer**: Comprehensive footer with social links and resources

## 🛠️ Tech Stack

- **React 18**: Latest React features and hooks
- **Tailwind CSS 3**: Utility-first styling
- **Framer Motion**: Advanced animations and interactions
- **Lucide React**: Beautiful SVG icons
- **Vite**: Lightning-fast build tool

## 📦 Installation

1. Navigate to the project directory:
```bash
cd /home/omkarsingh/Desktop/Projects/internship-web
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm run dev
```

4. Open browser to `http://localhost:3000`

## 🚀 Building for Production

```bash
npm run build
```

Build files will be in the `dist` directory.
