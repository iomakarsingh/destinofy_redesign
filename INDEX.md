## 💡 Tips & Tricks

### Performance
- Images load faster than emojis
- Use Tailwind utilities instead of custom CSS
- Component reuse improves maintainability

### Customization
- Keep color scheme consistent
- Use same spacing scale throughout
- Follow animation patterns

### Mobile First
- Always test on mobile
- Use viewport in index.html
- Test with real devices

## 🐛 Troubleshooting

### Common Issues & Solutions

**Port 3000 in use?**
```bash
npm run dev -- --port 3001
```

**Dependencies won't install?**
```bash
rm -rf node_modules package-lock.json
npm install
```

**Build failing?**
```bash
npm run build -- --debug
```
